About Section

About Sendvia

Sendvia is a fast, simple, and privacy-focused file sharing tool designed for direct transfers between people — no accounts, no ads, no storage.

Why did I create this?

Most file-sharing platforms are bloated, slow, or invasive. Sendvia was built to be lightweight, distraction-free, and focused on doing one thing well: sending files directly.

How does Sendvia work?

Files are transferred directly between the sender and receiver using peer-to-peer technology (WebRTC). A temporary code is only used to establish the connection.

Are my files stored on any server?

No. Files are never uploaded, stored, logged, or backed up. They move directly from sender to receiver.

  • No cloud storage
  • No file history
  • No access to your data

What data is stored?

Only minimal, temporary connection data (such as a transfer code and signaling information) is stored to help peers connect. This data is removed automatically after use.

Is Sendvia open source?

Yes. Sendvia is open source and transparent by design.

GitHub: View source code

Is Sendvia free?

Yes. Sendvia is completely free to use with no subscriptions or limits.

Is this secure?

Transfers are encrypted end-to-end using WebRTC's built-in security. Only the sender and receiver can access the files.

Does Sendvia track users?

No analytics, no ads, no tracking, and no user profiling.

Who made this?

Sendvia is developed and maintained independently with a focus on privacy, simplicity, and clean design.

Help Section

1. Start a transfer

Click "Drop a file to share" and select the file you want to send.

2. Select your file

Choose any file from your device to begin the upload.

3. Upload to the website

Wait while the file is securely prepared for sharing.

4. Copy the generated code

Once the upload is complete, copy the unique transfer code.

5. Use the QR option (optional)

Click "Show QR" to display a scannable QR code instead.

6. Share with the recipient

Send the code or link to the person you want to share the file with.

7. Or copy the link

You can also manually copy and share the link.

8. Enter the transfer code

Paste the received code on the website to access the file.

9. Download the file

Click "Download File" to receive the shared file.
